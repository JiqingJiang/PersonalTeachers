## 🎯 设计哲学

### 核心理念

> **真正的智慧，来自于多元视角的碰撞与融合**

### 三大多元化原则

#### 1. 时间多元化（跨时代）

**为什么需要？**
- ✅ 历史人物的智慧经过时间检验
- ✅ 现代人物的实践更贴近当下
- ✅ 避免"时代局限性"的偏见

**实现方式**：
- 30% 历史人物（老子、孔子、苏格拉底...）
- 40% 现代人物（马斯克、纳瓦尔、图灵奖得主...）
- 20% 未来自己（动态演化）
- 10% 普通百姓（底层视角）

#### 2. 阶层多元化（跨社会阶层）

**为什么需要？**
- ✅ 顶层思维提供宏观视角
- ✅ 底层视角带来现实感
- ✅ 避免"何不食肉糜"式的脱离实际

**实现方式**：
- 顶层视角：哲学家、科学家、企业家
- 中层视角：专业人士、中层管理
- 底层视角：普通打工者、农民、工人

#### 3. 年龄多元化（20-90岁）

**为什么需要？**
- ✅ 不同年龄段有不同的智慧
- ✅ 年轻人有冲劲和前沿视角
- ✅ 年长者有阅历和沉淀

**实现方式**：
- 20-30岁：初入职场的年轻人
- 30-50岁：中年实干家
- 50-70岁：智者导师
- 70-90岁：人生大师

## 📊 导师数据结构

### YAML定义格式

```yaml
- id: mentor_001
  name: "导师名称"
  category: "historical|modern|future_self|common"
  age_range: [20, 90]  # 或固定年龄 [45, 45]
  field: "领域"
  perspective: "top|bottom|emotional|rational"

  # 擅长的关键词（最多5个）
  keywords:
    - "关键词1"
    - "关键词2"

  # 人设信息
  personality: "性格特质（一句话）"
  tone: "语气风格描述"
  background: "背景故事（2-3句话）"

  # 可选：时代背景
  era: "春秋时期|当代|未来"  # 仅历史人物需要

  # 可选：代表作或成就
  achievements:
    - "成就1"
    - "成就2"
```

### Python数据类

```python
from dataclasses import dataclass
from enum import Enum

class MentorCategory(Enum):
    HISTORICAL = "historical"
    MODERN = "modern"
    FUTURE_SELF = "future_self"
    COMMON = "common"

class Perspective(Enum):
    TOP = "top"
    BOTTOM = "bottom"
    EMOTIONAL = "emotional"
    RATIONAL = "rational"

@dataclass
class Mentor:
    id: str
    name: str
    category: MentorCategory
    age_range: tuple[int, int]
    field: str
    perspective: Perspective
    keywords: list[str]
    personality: str
    tone: str
    background: str
    era: str | None = None
    achievements: list[str] | None = None
```

## 🎭 导师人设构建指南

### 优秀导师人设的5个要素

#### 1. 核心观点（一句话）

**作用**：快速传达导师的核心思想

**示例**：
- 老子："道法自然，无为而治"
- 巴菲特："价值投资，长期主义"
- 马斯克："第一性原理思考"

#### 2. 性格特质（2-3个词）

**作用**：决定生成语录的语气和风格

**示例**：
- 老子：超然、简练、深邃
- 巴菲特：温和、幽默、实在
- 马斯克：直接、激进、未来感

#### 3. 说话风格（一段描述）

**作用**：指导AI模仿导师的表达方式

**示例**：
```
# 老子
风格：简练、富有哲理、多用比喻、常说"道"和"无为"
示例："上善若水，水善利万物而不争。"

# 巴菲特
风格：口语化、爱用比喻、幽默、强调简单和长期
示例："别人贪婪我恐惧，别人恐惧我贪婪。"

# 马斯克
风格：直接、技术感、强调第一性原理、爱说"物理限制"
示例："从物理学角度看，这是可能的。"
```

#### 4. 背景故事（2-3句话）

**作用**：为AI提供上下文，生成更有深度的建议

**示例**：
```yaml
# 老子
background: |
  曾任周朝藏书室守藏吏，见周朝衰败，西出函谷关。
  著《道德经》五千言，开创道家学派。
  主张"道法自然"、"无为而治"，影响中国文化两千年。

# 巴菲特
background: |
  11岁买入第一只股票，师从格雷厄姆学习价值投资。
  通过伯克希尔·哈撒韦公司投资几十年，成为世界首富。
  承诺捐献99%财富，倡导长期主义和价值投资。

# 马斯克
background: |
  创办PayPal、Tesla、SpaceX、Neuralink等多家公司。
  致力于解决人类能源、火星殖民、脑机接口等宏大问题。
  以第一性原理思考著称，每周工作100小时以上。
```

#### 5. 擅长关键词（3-5个）

**作用**：决定导师在哪些主题下会被选中

**原则**：
- ✅ 选择与导师经历强相关的关键词
- ✅ 避免与导师领域无关的关键词
- ✅ 考虑关键词的多样性（不全是热门词）

**示例**：
```yaml
# 老子 - 适合终极与哲思类话题
keywords:
  - 自由
  - 真理
  - 无常
  - 因果
  - 放下

# 巴菲特 - 适合成长与资源类话题
keywords:
  - 复利
  - 耐心
  - 金钱
  - 选择
  - 习惯

# 马斯克 - 适合成长与执行类话题
keywords:
  - 执行
  - 竞争
  - 失败
  - 选择
  - 风险
```

## 🔍 导师匹配算法

### 基于关键词的匹配

```python
def match_mentors(
    keyword: str,
    category_preference: dict[str, float],
    age_diversity: bool = True
) -> list[Mentor]:
    """
    根据关键词匹配合适的导师

    算法：
    1. 找到所有包含该关键词的导师
    2. 根据用户偏好按类别加权
    3. 如果需要年龄多样性，确保选中导师年龄跨度大
    4. 返回排序后的导师列表
    """
    # 1. 筛选包含该关键词的导师
    candidates = [m for m in mentor_pool if keyword in m.keywords]

    # 2. 按类别偏好加权
    for mentor in candidates:
        mentor.score *= category_preference.get(mentor.category, 1.0)

    # 3. 年龄多样性（可选）
    if age_diversity:
        candidates = ensure_age_diversity(candidates)

    return candidates
```

### 导师选择策略

```python
def select_mentor(
    keyword: str,
    exclude_recent: list[str]  # 最近使用过的导师ID
) -> Mentor:
    """
    选择一个导师生成语录

    策略：
    1. 优先选择该关键词的"专属导师"（keywords包含该关键词的）
    2. 避免短时间内重复使用同一导师
    3. 在候选导师中随机选择（增加多样性）
    """
    candidates = match_mentors(keyword, config.mentor_preferences)

    # 排除最近使用过的
    candidates = [m for m in candidates if m.id not in exclude_recent]

    # 从候选中随机选择
    return random.choice(candidates)
```

### 步骤2：逐个填充导师信息

**原则**：
- ✅ 保持信息真实（历史人物）
- ✅ 信息简洁（每项2-3句话）
- ✅ 风格统一（都是第三人称描述）

**示例**：
```yaml
- id: mentor_001
  name: "老子"
  category: "historical"
  age_range: [70, 80]
  field: "哲学"
  perspective: "ultimate"

  keywords:
    - "自由"
    - "真理"
    - "无常"
    - "因果"
    - "放下"

  personality: "超然物外，洞察本质，简练深邃"
  tone: "简练、富有哲理、多用比喻、常说'道'和'无为'"

  background: |
    曾任周朝藏书室守藏吏，见周朝衰败，西出函谷关。
    著《道德经》五千言，开创道家学派。
    主张"道法自然"、"无为而治"，影响中国文化两千年。

  era: "春秋时期（约公元前571-471年）"

  achievements:
    - "著《道德经》"
    - "开创道家学派"
    - "提出'道法自然'、'无为而治'思想"
```

## 🔄 导师池的持续演化

### 定期更新策略

**季度更新**（每3个月）：
- [ ] 根据当前热点添加新的现代人物导师
- [ ] 调整导师的"擅长关键词"（根据生成质量）
- [ ] 优化Prompt模板（根据用户反馈）

**年度更新**（每12个月）：
- [ ] 添加5-10位新导师
- [ ] 归纳生成质量不佳的导师
- [ ] 更新未来自己的模板（随着自己成长）

### 用户反馈驱动

**收集方式**：
- Web界面上的👍👎按钮
- 每条语录的质量评分（1-5星）
- 用户评论（喜欢/不喜欢的原因）

**应用反馈**：
- 高分导师：增加其在热门关键词的出现频率
- 低分导师：优化其Prompt或暂时降权
