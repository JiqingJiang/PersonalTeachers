**检查方法**：
- 登录QQ邮箱网页版
- 查看"垃圾邮件"文件夹
- 查看"已删除"文件夹

**解决方法**：
- 将发件人 `erode1701@qq.com` 添加到白名单
- 设置邮件规则：将来自自己的邮件移到收件箱

### 2. QQ邮箱 SMTP服务限制 ⚠️
QQ邮箱免费版SMTP有限制：

**限制条件**：
- 每天发送数量限制（通常200-500封）
- 单位时间频率限制
- 相同内容可能被识别为群发

**验证方法**：
```bash
# 手动发送一封简单测试邮件
curl -X POST http://erode.site:8000/api/email/test \
  -H "Content-Type: application/json" \
  -d '{"to_email": "erode1701@qq.com"}'
```

### 3. 邮件延迟 ⚠️
QQ邮箱可能有延迟：
- 正常延迟：1-5分钟
- 高峰期延迟：10-30分钟

## 🛠️ 解决方案

### 方案1：检查QQ邮箱设置

1. 登录QQ邮箱：https://mail.qq.com
2. 点击"设置" → "反垃圾"
3. 检查是否有拦截记录
4. 添加白名单：erode1701@qq.com

### 方案2：更换SMTP服务器

如果QQ邮箱继续有问题，建议使用其他邮箱服务：

**Gmail（推荐）**：
```bash
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USERNAME=your-email@gmail.com
SMTP_PASSWORD=应用专用密码
```

**126/163邮箱**：
```bash
SMTP_HOST=smtp.126.com  # 或 smtp.163.com
SMTP_PORT=465
SMTP_USERNAME=your-email@126.com
SMTP_PASSWORD=授权码
```
